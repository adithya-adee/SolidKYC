# SolidKYC Backend

TypeScript/Node.js backend for verifying ZK proofs generated by snarkjs.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Run in development mode with hot-reload:
```bash
npm run dev
```

## Build

Build the TypeScript code:
```bash
npm run build
```

## Production

Run the built code:
```bash
npm start
```

## Test Verification

Test the verification with existing proof.json and public.json:
```bash
npx ts-node src/verifier.ts
```

## API Endpoints

### GET /health
Health check endpoint.

**Response:**
```json
{
  "status": "healthy",
  "timestamp": 1234567890
}
```

### POST /verify
Verify a Groth16 proof.

**Request body:**
```json
{
  "proof": {
    "pi_a": ["...", "...", "..."],
    "pi_b": [["...", "..."], ["...", "..."], ["...", "..."]],
    "pi_c": ["...", "...", "..."],
    "protocol": "groth16",
    "curve": "bn128"
  },
  "public": ["0", "1765032625", "1760000000", "...", "..."]
}
```

**Response:**
```json
{
  "verified": true,
  "error": null
}
```

## Files

- `proof.json` - Example proof from snarkjs
- `public.json` - Example public inputs
- `public/verification_key.json` - Verification key for the circuit
